# Overview

## Product Vision

Job Tracker is a web application that helps job seekers **organize applications**, **track responses**, **manage interviews, offers, and ghosting**, and **extract skill trends** from job postings.

The goal is to reduce the chaos of job searching and transform the process into something clear, measurable, and strategic.

## Problem

People actively looking for work — recent graduates, junior professionals, and career changers — send many CVs per day and end up:

- Losing track of where they applied
- Not knowing who responded and who didn't
- Forgetting follow-ups
- Having no visibility into their pipeline (interviews, offers, rejections, ghosting)
- Not understanding market patterns (most requested skills, growing tech stacks)

The result: anxiety, disorganization, and reduced effectiveness in the job search.

## Core Value Proposition

**Primary value:** "A simple and practical way to control all applications in one place."

**Differentiating value:** "Beyond tracking, the app shows real trends of skills requested in job postings."

## Product Objectives

### User Objectives

- Quickly register job applications
- See the current status of each application
- Never forget follow-ups
- Track interviews and offers
- Understand which skills are most requested in saved postings

### Product Objectives

- Create a daily usage habit
- Deliver value with minimal input effort
- Provide immediate value (organization + insights)

## Non-Goals (Out of MVP Scope)

To keep the MVP focused, the following are **explicitly excluded**:

- AI-powered CV/cover letter generation
- Automatic job applications
- Full LinkedIn/Indeed/etc. API integration
- Advanced networking CRM
- Complex fit scoring
- Native mobile app (start with responsive web)
- Multi-user collaboration

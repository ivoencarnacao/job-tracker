# User Stories

## Tracking

| ID | Priority | Story |
|----|----------|-------|
| US-01 | Must | As a job seeker, I want to quickly register a new application so that I don't forget where I applied. |
| US-02 | Must | As a job seeker, I want to update the status of an application so that I know what stage each process is at. |
| US-03 | Must | As a job seeker, I want to filter applications by status so that I can focus on interviews, offers, or rejections separately. |

## Follow-ups

| ID | Priority | Story |
|----|----------|-------|
| US-04 | Must | As a job seeker, I want to set a follow-up reminder on an application so that I don't let opportunities die. |
| US-05 | Should | As a job seeker, I want to see applications with no response for a long time so that I can decide whether to follow up or mark them as ghosted. |

## Interviews and Offers

| ID | Priority | Story |
|----|----------|-------|
| US-06 | Must | As a job seeker, I want to record interviews with date, type, and notes so that I can prepare better. |
| US-07 | Should | As a job seeker, I want to record offers and compare them in my notes so that I can make informed decisions. |

## Skills and Trends

| ID | Priority | Story |
|----|----------|-------|
| US-08 | Should | As a job seeker, I want to see which skills appear most often in my saved job postings so that I can adjust my CV and study plan. |
| US-09 | Should | As a career changer, I want to understand which tech stack is most requested so that I can focus my learning on what matters. |

## Dashboard

| ID | Priority | Story |
|----|----------|-------|
| US-10 | Must | As a job seeker, I want to see an overview dashboard with key metrics so that I understand my job search progress at a glance. |

## Search and Filtering

| ID | Priority | Story |
|----|----------|-------|
| US-11 | Must | As a job seeker, I want to search applications by company or job title so that I can quickly find a specific application. |
| US-12 | Should | As a job seeker, I want to sort applications by date or last update so that I can prioritize recent activity. |
